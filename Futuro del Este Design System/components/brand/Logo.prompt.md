The Futuro del Este logo — circular wave mark plus wordmark. The mark is embedded, so it renders anywhere with no asset wiring.

```jsx
<Logo variant="lockup" size={56} />
<Logo variant="mark" size={40} />
<Logo variant="lockup" tone="white" />   {/* on brand/photo backgrounds */}
```

Variants: `mark` (just the wave), `lockup` (mark + stacked FUNDACIÓN / FUTURODELESTE), `lockup-horizontal`. `tone="white"` for dark/photo backgrounds. `size` drives the whole lockup.
