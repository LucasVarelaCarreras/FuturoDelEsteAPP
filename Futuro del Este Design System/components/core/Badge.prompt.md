Small pill label for status, category, date or count chips.

```jsx
<Badge tone="emerald">Sustentabilidad</Badge>
<Badge tone="cyan" dot>Inscripción abierta</Badge>
<Badge tone="ocean" soft={false}>13 JUN</Badge>
```

Tones: `cyan`, `emerald`, `aqua`, `ocean`, `navy`, `success`, `warning`. `soft` (default) = tinted; `soft={false}` = solid brand fill with white text. `dot` adds a leading status dot.
