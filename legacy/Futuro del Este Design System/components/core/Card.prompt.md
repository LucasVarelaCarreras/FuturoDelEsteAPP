Soft rounded surface — the base container for features, news, stats and panels.

```jsx
<Card accent interactive padding="lg">
  <Badge tone="emerald">Novedades</Badge>
  <h3>Día Mundial del Medio Ambiente</h3>
  <p>Actividades de concientización en Punta del Este.</p>
</Card>
```

Props: `padding` (`none`/`sm`/`md`/`lg`), `accent` (wave gradient top strip), `interactive` (lift + shadow on hover). Compose freely with Badge, headings and Button inside.
