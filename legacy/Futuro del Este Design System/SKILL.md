Read the `readme.md` file within this skill, and explore the other available files
(`styles.css` + `tokens/`, `components/`, `ui_kits/`, `social/`, `guidelines/`, `assets/`).

If creating visual artifacts (slides, mocks, social posts, throwaway prototypes, etc),
copy assets out (notably `assets/logo-mark.png` and `assets/hero-web.png`) and create
static HTML files for the user to view, linking `styles.css` for the brand tokens.
If working on production code, copy assets and read the rules here to become an expert
in designing with this brand.

Key brand facts: voseo Spanish (Sumate, Colaborá), warm/inclusive "nosotros" voice,
the ocean-wave motif, cyan `#019AC4` primary on a teal→ocean ramp, pill buttons, soft
rounded cards, Oswald (display) / Mulish (UI) / Lora-italic (accent) type. Fonts and
the icon set are nearest-match substitutions — see the caveats in `readme.md`.

If the user invokes this skill without any other guidance, ask them what they want to
build or design, ask some questions, and act as an expert designer who outputs HTML
artifacts _or_ production code, depending on the need.
